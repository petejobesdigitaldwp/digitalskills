This is the prototype for the DWP Churchill project. This is a data tool to allow users to analyse local data in context and help with decision making.

Prototypes

Version 0.5 (Place and Subject of Interest Pages, initial integration)

Adding the option to see what makes a place or a dataset particularly interesting. This needs fleshing out in terms of data, but would highlight outliers or extremes in the data.

Note: Select boxes for location would be based on GOV.UK Location Picker w/ AutoComplete. Date entry on the 'when' page may have to use select box as the number of dates that can be chosen are limited to a defined data set.

Version 0.4 (Broader proposition)

Broadening out the proposition so that the user chooses whether to look primarily for data about a place, or about a subject.

Note: Select boxes for location would be based on GOV.UK Location Picker w/ AutoComplete. Date entry on the 'when' page may have to use select box as the number of dates that can be chosen are limited to a defined data set.

Version 0.3

Experimenting with a secondary navigation at the top of the page once viewing charts.

Version 0.2

Experimenting with drop down navigation at the top of the page once viewing charts, closer to David's new nav prototype but using an initial process to get into the main stats page.

Version 0.1

Initial prototype designed by Pete in his first week. Aiming to simplify user journey and navigation out as much as possible without losing functionality. GDS styles have been used where possible.

New features include large text stats on snapshot and a table view of data for better accessibility.

Note: forms would use predictive input rather than select box.
